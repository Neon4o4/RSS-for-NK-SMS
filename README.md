南开数院的非官方新闻RSS
=======================
数院官网上有各种新闻，但是由于更新频率不太高，所以经常手动去刷新比较烦，收益还很小。
于是出现了这个非官方的RSS。用户订阅RSS之后，根据自己RSS阅读器的设置可以定时的更新新闻。

工作原理：
=======================
**.xml：分类保存新闻的数据，供RSS阅读器更新。

RssSync.py：在服务器端定时在数院网站抓取数据，发现官网有更新便立即更新xml文件。

latest：保存上次更新时每个分类最新一篇文章的链接。

大概就是这样。
欢迎各位制作其他学院的版本~
